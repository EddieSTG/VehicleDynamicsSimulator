Project Core & Objectives

This simulator aims to achieve maximum physical accuracy by computing complete chassis, wheel, and suspension mechanics in real time. It implements a dedicated high-frequency physics integration routine decoupled from the variable graphics rendering pipeline, allowing accurate longitudinal validation of performance statistics.

High-Fidelity Physics

Runge-Kutta 4th Order (RK4) numerical integration solves stiff differential equations at 1000Hz, ensuring absolute stability for sudden tire spikes and high suspension rates.

Live Telemetry

Real-time data visualization utilizing Dear ImGui and ImPlot overlays key vehicle state parameters directly within the 3D application frame.

State Validation

A continuous snapshot buffer records the full 14 DOF state history vector, enabling real-time pause/rewind replays and raw CSV exporting for analytical validation.

Physics & Math Architecture

The vehicle state space is divided into 14 distinct degrees of freedom to capture complex dynamic transients during heavy longitudinal transitions.

06

Chassis DOFs

Translation (X, Y, Z) and Rotation (Roll, Pitch, Yaw) defining the rigid body chassis kinematics.

04

Suspension DOFs

Independent vertical displacement for each wheel assembly relative to the vehicle base mass.

04

Wheel Rotation DOFs

Independent angular spin velocity tracking for traction torque and brake force slip processing.

Aerodynamic Optimization Model

The aerodynamic drag acts against structural acceleration, introducing an active dynamic pitching moment based on the variance between center of gravity and pressure:

Fd = ½ · ρ · v² · Cd · A

Where ρ represents atmospheric air density (1.225 kg/m³), Cd is the specialized ultra-low drag coefficient (0.27), and A indicates the vehicle frontal cross-section area (2.1 m²).

Target Profile: VW Polo BlueMotion

The engine map, gearing metrics, and weight profiles are closely modeled after a 2015 Volkswagen Polo 1.4 TDI BlueMotion to match rigorous real-world validation baselines.

Software Architecture & Tech Stack

Core Platform

Native C++ compiled for x64 architecture via Visual Studio 2022 on Windows systems.

Graphics Engine

Modern OpenGL (Version 3.3 Core Profile) running customized vertex and fragment shaders for vector lines.

Context & Matrix Math

GLFW for input window management, GLAD as the extension function loader, and GLM for linear algebra.

Package Management

Microsoft vcpkg architecture pipelines to ensure clean, repeatable multi-environment builds.
